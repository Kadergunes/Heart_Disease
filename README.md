# Heart_Disease
This project predicts heart disease using the UCI Heart Disease dataset and the K-Nearest Neighbors (KNN) algorithm.
Missing values handled with median and mode
Categorical features encoded using One-Hot Encoding
Technologies: Python, Pandas, Scikit-learn
